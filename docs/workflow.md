# Nomi Workflow

Nomi connects four production steps:

```txt
Script -> Image / Video Generation -> Timeline Editing -> Preview / Export
```

## Script To Nodes

Script text can become image or video node suggestions. You choose which suggestions become real nodes.

## Nodes To Timeline

Generated images and videos can be dragged into the timeline as clips.

## Timeline To Canvas

Edited clips can become references for another generation pass.

## Agent Role

The agent can suggest structure, prompts, and next actions. It should not hide failures, auto-select final assets, or replace creator judgment.
